# 3D Trail Map

A cross-platform app that displays a 3D topographic view of a mountain with hiking trails. Built with React, React Native, and react-three-fiber.

## Projects

### Web (`web/`)

React web app using Vite and react-three-fiber.

```bash
cd web
npm install
npm run dev
```

### Mobile (`mobile/`)

React Native app using Expo and react-three-fiber with expo-gl.

```bash
cd mobile
npm install
npx expo start
```

Then press `i` for iOS simulator or `a` for Android emulator.
